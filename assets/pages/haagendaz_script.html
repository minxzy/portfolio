영상 삽입과 슬라이딩 효과 적용

HTML 전체에 적용하는 border값 오류

HTML 자체에 border값이 적용되지 않아 다양한 시도를 해보게 되었습니다. HTML 전체를 감싸는 container를 만들어 브라우저에 대응하는 너비와 높이값을 주어 큰 틀을 제작 후 양측에 solid값을 부여했지만 구현되지 않았습니다. 결국 div에 하나씩 container를 부여해 개별로 값을 주어 문제를 해결 할 수 있었습니다.
