Fullpage.js를 활용한 레이아웃

Javascript Plugin인 Fullpage.js를 통해 스크롤시 한 개의 section씩 화면을 구현시키는 방법을 공부할 수 있게 되었습니다.

움직이는 Background와 Animation

YT player.js와 video, iframe 태그를 활용해 내부에서 불러온 mp4 파일과 유튜브를 background에 배치하는 방법을 공부할 수 있게 되었습니다. 처음에는 무작정 css body 태그에 유튜브에서 영상에 부여하는 고유 공유형 iframe 태그를 연동하여 영상을 로드하려 했지만 제대로 작동하지 않았습니다. 이후 Plugin과 따로 mp4 파일을 직접 저장하여 로드함으로 인해 문제를 해결 할 수 있게 되었습니다.

카카오 API를 통한 로컬 API 서비스

KAKAO에서 배포하는 로컬 API를 통해 페이지 내 원하는 크기의 지도를 불러오고, KAKAO에서 재공하는 위,경도의 좌표를 활용해 지정하고자 하는 위치에 마커를 표시하는 방법을 공부할 수 있게 되었습니다.
